# Robot Visualization Frontend

This is the React frontend for the robot visualization app. Uses Vite, React, leaflet, and react-leaflet.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Features
- Map of Downtown LA using leaflet
- Robot markers update in real time from backend
- Controls for move/reset/start/stop
